Ценообразование

Глобальный коэффициент наценки

Формула расчёта цены

Цена = (объём_см³ × цена_за_см³ + сложность) × коэффициент

• Объём — размер заготовки в кубических сантиметрах
• Цена за см³ — стоимость материала за 1 см³
• Сложность — фиксированная надбавка за работу (₽)
• Коэффициент — глобальный множитель (задаётся здесь)
Примеры: 1.0 = без наценки · 1.5 = +50% · 2.0 = ×2
Меняете здесь → все цены пересчитываются автоматически.

Безопасность

Состояние защиты панели управления

Меры защиты

Хэширование пароля (SHA-256)Пароль не хранится в открытом виде — только его хэш.
Rate-limit: 5 попыток → блокировка на 15 минутЗащита от перебора паролей (brute-force).
XSS-защита: HTML-экранирование данныхВсе данные из базы экранируются перед вставкой в DOM.
Проверка типа и размера загружаемых фотоПринимаются только JPG/PNG/WebP до 5 МБ.
Content-Security-PolicyЗапрет загрузки скриптов с посторонних доменов.
Правила Firestore — требуют настройки!В тестовом режиме запись открыта всем. После отладки установите правила: write только для авторизованных пользователей Firebase Auth.
Смена пароляЗамените ADMIN_HASH в начале файла. Сгенерировать SHA-256: emn178.github.io/online-tools/sha256.html

Как подключить Firebase

1. Зайдите на console.firebase.google.com → «Создать проект».

2. Firestore Database → Создать → Режим «Тест».

3. Storage → Начать → Режим «Тест».

4. Настройки проекта → Ваши приложения → «</>» → скопируйте firebaseConfig.

5. Вставьте конфиг в начало этого файла (admin.html) и в firebase-loader.js.
